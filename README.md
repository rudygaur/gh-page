# Personal resume site (GitHub Pages)

This repository is a **small static website** that publishes a single-page résumé. It is meant to be hosted on **GitHub Pages** so the CV is always available at a public URL without PDFs or external builders.

The résumé copy and layout live in **`index.html`**. The page uses plain **HTML and CSS** (no framework or build step): a responsive two-column layout on desktop, a single column on small screens, and print-friendly styles so it can be saved or printed cleanly.

To view it locally, open `index.html` in a browser or serve this folder with any static file server.
